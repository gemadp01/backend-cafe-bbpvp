## Run Locally

Clone the project

```bash
  git clone https://github.com/gemadp01/backend-cafe-bbpvp.git
```

Go to the project directory

```bash
  cd backend-cafe-bbpvp
```

copy .env from .env.example (for custom token)

```bash
  cp .env.example .env
```

set your costum token

```bash
  ACCESS_TOKEN_SECRET=[your-custom-token-here]
```

create folder for storing images

```bash
  mkdir -p public/uploads
```

Install dependencies

```bash
  npm install
```

Start the server
```bash
  npm run dev
```
